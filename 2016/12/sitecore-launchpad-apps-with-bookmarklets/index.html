Sitecore Launchpad apps and tools made easy with Bookmarklets

December 23, 2016 in Sitecore · 3 minute read

If you have worked with Sitecore for a while, in the version 6 or earlier days, you would remember that we didn’t have the luxury of Launchpad back then. You would spend most of the time in the desktop mode because that’s how you access most of the apps, from the start menu. Or the desktop shortcuts, which I almost never used, and I doubt most people even knew of. The biggest problem with the desktop mode (apart from being a memory hog) was, and it still is, that you would lose the state of the opened windows if you refresh the browser. And as developers, that’s what we do most of the time right.

Sitecore 6 Desktop

Fast forward to 7.1 / 7.2, but really the version 8 and we have now a very stable and handy Launchpad. And as the name suggests, it is the launch pad for the commonly used sitecore applications, and also the new modern SPEAK apps. So you don’t have to rely much on the desktop mode now.

Sitecore 8 Launchpad

However in saying that, now you need to come back to the Launchpad whenever you need to switch to a different app. This is an extra page load inbetween switching apps. This is still far from a painful experience but I would definitely prefer to cut this “middle” page out. I think It is the laziness in me (or my impatience?) that drives me to look for quicker ways to improvise such tedious workflows. And so the hunt began.

The Find

I found a chrome extension called Sitecore Developer Tool that gives you nice little shortcuts to most common pages/apps in Sitecore. Exactly what I wanted, this tool works with the sitecore url in the current browser tab and lets you quickly switch between admin pages, apps, databases and modes without having to go to the Launchpad or Desktop mode or type the url. Great!!!

The Bad

This may not be a problem for you if you primarily use Chrome for Sitecore, but lately I have been using Firefox for everything Sitecore. So the biggest drawback for me now is this tool does not have a Firefox counterpart. And I couldn’t find any other equivalent. So I had to come up with something.

The Bookmarklets

The easy solution was to write bookmarklets. To achieve a similar functionality, I created bookmarklets to most of the common sitecore apps that are accessible from the launchpad and also the admin pages and other tools. These bookmarklets are basically normal bookmarks with scripts to calculate the context site url, hence very light-weight and works with most popular browsers.

A bookmarklet is a bookmark stored in a web browser that contains JavaScript commands that add new features to the browser. Bookmarklets are unobtrusive JavaScripts stored as the URL of a bookmark in a web browser or as a hyperlink on a web page. Read more here

The advantages:

  • simple and light-weight
  • works on any sitecore url
  • works on most major browsers
  • can be synced between different browsers and machines using tools like Xmarks Bookmark Sync
  • easily editable and extendable as they are just bookmarks

Sitecore Launchpad Bookmarkets in Chrome
Sitecore Launchpad Bookmarkets in Firefox

How to Install

  1. Download the bookmarks HTML file from the gist below.
  2. Import the HTML file into your browser via its bookmark manager.

You will see a bookmarks folder called Sitecore Launchpad being imported to the browser’s default import location. Feel free to move it where you want.
